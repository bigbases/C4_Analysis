# LLM-based Annotation

Automated annotation system using ChatGPT and Claude with multiple personas.

## Files
- `gpt_annotation.py` - Main annotation orchestrator
- `chatgpt/chatgpt_request.py` - ChatGPT API integration
- `claude/claude_request.py` - Claude API integration
- `prompt/` - Role-based prompt templates

## Usage
```bash
python gpt_annotation.py
```
